How Performance Is Calculated

We look at every run of a given specialization and assign:

  • Positive points when the key upgrades (higher-level keys earn more points).
  • Negative points when the key depletes (losing a low-level key is judged more harshly).

Then we average those points across all runs to get a single “Performance” number.

Why This Matters

  • Rewards high-level pushes more strongly.
  • Punishes depletes in a balanced way.
  • Gives a fair comparison regardless of how many runs each spec has.

Summary

This chart helps you spot specializations that are both strong in pushing keys (high performance) and popular among players (many runs).
